# AI Chatbot with NLP

## Project Description

This project implements a simple AI chatbot using Python and the Natural Language Processing (NLP) library NLTK. The chatbot can understand user queries and respond with predefined answers.

## Technologies Used

Python
NLTK (Natural Language Toolkit)

## Features

* Basic natural language processing
* Tokenization using NLTK
* Interactive chatbot conversation
* Predefined responses to user queries

## Installation

Install the required library:

pip install nltk

## How to Run

Run the chatbot script:

python chatbot.py

The chatbot will start and respond to user queries in the terminal.

## Example Queries

hello
how are you
your name
